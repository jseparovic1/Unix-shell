# Unix-shell
My unix shell

# Installation
```
$ make
$ ./jusepShell
```

# Avalible shell commands

Built in commands :
```
$ help
$ cd
$ kill
$ clear
$ exit
```

# Extra
Redirection
```
$ ls -al > output.txt
```
Pipes
```
$ cat numbers.txt | sort
```

Shell also supports any other unix command.
