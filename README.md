# Unix-shell
Simple unix shell

# Installation
```
$ make
$ ./jusepShell
```

# Avalible shell commands

Built in commands :
```
$ ./jusepShell help
$ ./jusepShell cd
$ ./jusepShell kill
$ ./jusepShell clear
$ ./jusepShell exit
```

# Extra
Redirection
```
$ ./jusepShell ls -al > output.txt
```
Pipes
```
$ ./jusepShell cat numbers.txt | sort
```

Shell also supports any other unix command.
